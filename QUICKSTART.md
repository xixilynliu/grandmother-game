## 🎮 游戏操作指南

### 基本操作
1. **阅读对话**：场景会自动展示角色对话
2. **做出选择**：点击底部的选择按钮
3. **查看属性**：右侧显示当前角色属性和关系
4. **继续游戏**：选择后自动跳转到下一场景

### 选择提示
- 🔒 **灰色按钮**：属性不足，无法选择
- 💫 **高亮选项**：推荐选择或特殊选项
- 📊 **属性要求**：鼠标悬停查看需要的属性值

## 💾 存档系统

游戏使用 `localStorage` 自动保存进度：
- 每次选择后自动保存
- 刷新页面后继续上次进度
- 可在开发者工具中清除存档

```javascript
// 清除存档（浏览器控制台）
localStorage.clear()
location.reload()
```

## 🛠️ 开发者选项

### 查看当前游戏状态
```javascript
// 浏览器控制台
console.log(localStorage.getItem('persist:root'))
```

### 修改属性值（测试用）
```javascript
// 示例：提升所有属性到80
const state = JSON.parse(localStorage.getItem('persist:root'))
const game = JSON.parse(state.game)
game.playerStats = {
  wisdom: 80,
  charisma: 80,
  modernity: 80,
  teaching: 80,
  medical: 80
}
state.game = JSON.stringify(game)
localStorage.setItem('persist:root', JSON.stringify(state))
location.reload()
```
